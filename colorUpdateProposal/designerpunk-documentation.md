## Typography

### Font Pairings
DesignerPunk uses a carefully curated set of fonts to achieve its cyberpunk aesthetic. All fonts are **open source** under the SIL Open Font License.

#### Recommended Primary Stack
- **Display Headers**: Rajdhani Bold or Orbitron
- **Body Text & UI**: Inter Regular/Medium
- **Code/Data/Technical**: Share Tech Mono

### Display / Header Fonts

#### Orbitron
- **License**: Open Source (SIL Open Font License)
- **Use Case**: Headers, UI elements, futuristic branding
- **Aesthetic**: Geometric sans-serif with clean, tech-forward lines
- **Best For**: Main headers, logo, primary CTAs

#### Rajdhani
- **License**: Open Source (SIL Open Font License)
- **Use Case**: Headers, subheadings, navigation
- **Aesthetic**: Angular, condensed, industrial edge
- **Best For**: Section headers, card titles, UI labels
- **Pairs Best With**: Inter (top recommendation)

#### Saira
- **License**: Open Source (SIL Open Font License)
- **Use Case**: Display text, feature headers
- **Aesthetic**: Semi-condensed with sharp terminals, slightly glitchy
- **Best For**: Feature announcements, modal headers

#### Exo 2
- **License**: Open Source (SIL Open Font License)
- **Use Case**: Headers, UI elements
- **Aesthetic**: Technological, modern, geometric construction
- **Best For**: Dashboard headers, data visualization titles

#### Teko
- **License**: Open Source (SIL Open Font License)
- **Use Case**: Compact headers, data displays
- **Aesthetic**: Extremely condensed and angular
- **Best For**: UI labels, compact information, data-dense interfaces
- **Note**: Use sparingly for maximum impact

## Color Theory

### Split-Complementary Foundation
The palette is built on a split-complementary color scheme using four primary accents:
- **Yellow (#F9F002)** - High energy, attention-grabbing
- **Purple (#B026FF)** - Brand identity, sophistication  
- **Cyan (#00F0FF)** - Technology, contrast
- **Electric Green (#00FF41)** - Success, data, toxic aesthetic

These colors sit at strategic points on the color wheel, creating vibrant contrast while maintaining visual harmony. This forms the foundation of the cyberpunk aesthetic.

### Analogous Extensions
Each primary accent has an analogous partner that shares similar characteristics but with modified temperature:
- **Yellow → Amber** - Shifts toward red, adds warmth
- **Purple → Hot Pink** - Shifts toward red, adds aggressive neon edge
- **Cyan → Teal** - Adds green, grounds the palette
- **Electric Green** - Bridges cyan and yellow naturally, needs no partner

### Intensity Scale Philosophy
Each color family follows a consistent intensity progression:
- **100 (Lightest Tint)** - Color + significant white, 20-30% opacity feel
- **300 (Light Tint)** - Color + moderate white, 50-60% opacity feel  
- **500 (Base)** - Pure saturated color, 100% intensity
- **700 (Dark Shade)** - Color + moderate black, maintains vibrancy
- **900 (Darkest Shade)** - Color + significant black, anchoring tone

This scale provides flexibility for hierarchy, states, and accessibility while maintaining the vibrant cyberpunk aesthetic.

## Best Practices

### Color Usage Guidelines

1. **Use vibrant colors intentionally** - Not every element needs maximum saturation
2. **Leverage the intensity scale** - Use 300/700 variants for states and hierarchy
3. **Balance with strong neutrals** - Deep blues, grays, and true black are essential
4. **Maintain accessibility** - Ensure sufficient contrast ratios for text
5. **Consider context** - Adjust color choices based on content density and user task
6. **Don't use all colors at once** - Strategic restraint creates more impact

### Typography Guidelines

1. **Establish clear hierarchy** - Use font weight and size before adding more fonts
2. **Limit font families per screen** - Typically 2-3 fonts maximum
3. **Use monospace intentionally** - Share Tech Mono for technical content only
4. **Consider readability** - Inter for long-form content, decorative fonts sparingly
5. **Maintain consistency** - Define and stick to your heading/body combinations
6. **Test at scale** - Ensure fonts work at both large display and small UI sizes

### Accessibility Considerations

- **Contrast ratios**: Maintain WCAG AA standards (4.5:1 for normal text, 3:1 for large)
- **Color blindness**: Don't rely solely on color to convey information
- **Font sizes**: Minimum 16px for body text, larger for older audiences
- **Interactive states**: Provide clear hover, focus, and active states
- **Testing**: Validate with tools like WebAIM's contrast checker
