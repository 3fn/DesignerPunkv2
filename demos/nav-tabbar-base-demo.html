4. Accessibility Features

ARIA Attributes

  • role="tablist" on the container
  • role="tab" on each tab item
  • aria-selected="true|false" on each tab
  • aria-label from accessibilityLabel (not icon name)

Keyboard Navigation

Arrow keys: Left/Right arrows move focus between tabs with wrapping. Enter/Space selects the focused tab. Tab key enters on selected tab, Tab again exits.

Reduced Motion

Animation suppression: When prefers-reduced-motion: reduce is active, all animation phases collapse to instant state change. Initial render always snaps without animation.

5. Token Verification

Check browser DevTools to verify these CSS custom properties are loaded:

Color Tokens

  • --color-structure-canvas — container background
  • --color-structure-border-subtle — top stroke
  • --color-action-navigation — active icon + dot
  • --color-icon-navigation-inactive — inactive icon
  • --color-background-primary-subtle — glow center (active)

Spacing Tokens

  • --space-050 — dot size, active bottom padding
  • --space-100 — inactive bottom padding
  • --space-150 — active top/inline padding
  • --space-200 — inactive top padding, container inline margins

Motion Tokens

  • --motion-duration-150 — depart/arrive phase timing
  • --motion-duration-350 — dot glide timing
  • --motion-easing-glide-decelerate — dot glide curve

Blend Tokens

  • --blend-pressed-lighter — inactive tab press feedback (12% lighter)

6. Usage Examples

HTML

<nav-tab-bar selected-value="home"></nav-tab-bar>

<script>
  const tabBar = document.querySelector('nav-tab-bar');
  tabBar.tabs = [
    { value: 'home', icon: 'home-outline', activeIcon: 'home-solid', accessibilityLabel: 'Home' },
    { value: 'search', icon: 'search-outline', activeIcon: 'search-solid', accessibilityLabel: 'Search' },
    { value: 'profile', icon: 'person-outline', activeIcon: 'person-solid', accessibilityLabel: 'Profile' }
  ];
  tabBar.onSelectionChange = (value) => console.log('Selected:', value);
</script>

JavaScript

const tabBar = document.querySelector('nav-tab-bar');

// Listen for selection changes
tabBar.addEventListener('selection-change', (event) => {
  console.log('Selected:', event.detail.value);
});

// Programmatic selection
tabBar.selectedValue = 'search';
