2. Size Variants

Icon sizes are mathematically derived from the typography scale using fontSize √ó lineHeight. Each size aligns with a specific typography context.

Core sizes (90% of use cases): 20px, 24px, 32px, 36px, 40px. The remaining sizes cover edge cases like captions (13px) and display text (48px).

3. Color Variants

Icons support three color modes: inheritance from parent (default), design token references, and direct hex values.

Color Inheritance (Default)

When no color attribute is set, icons use currentColor and inherit from their parent element.

Parent: cyan-300
Parent: pink-300
Parent: purple-200

Token References

Pass a token name to the color attribute. It resolves to var(--token-name) internally.

color-feedback-success-text
color-feedback-error-text
color-feedback-info-text

Direct Hex Colors

Pass a hex value directly for one-off color overrides.

#FF6B9D
#4ECDC4
#FFE66D

4. Optical Balance

When optical-balance="true" is set alongside a hex color, the icon color is lightened by 8% using blend utilities. This compensates for icons appearing visually heavier than adjacent text due to stroke density.

Compare the icon weight ‚Äî the balanced version appears lighter to match text weight:

Without balance
With balance (8% lighter)
White without balance
White with balance
When to use: Optical balance is most useful when pairing icons with text labels in buttons, badges, or list items. It requires a hex color value ‚Äî it has no effect with inherit or token references.

7. Usage Examples

HTML

<!-- Basic icon (size is required) -->
<icon-base name="arrow-right" size="24"></icon-base>

<!-- With token color -->
<icon-base name="check" size="24" color="color-feedback-success-text"></icon-base>

<!-- With hex color and optical balance -->
<icon-base name="mail" size="32" color="#A855F7" optical-balance="true"></icon-base>

<!-- With test ID for automated testing -->
<icon-base name="settings" size="24" test-id="settings-icon"></icon-base>

<!-- Color inheritance from parent -->
<div style="color: var(--cyan-300);">
  <icon-base name="info" size="24"></icon-base>
  <span>Inherits cyan color</span>
</div>

JavaScript

// Programmatic creation
const icon = document.createElement('icon-base');
icon.name = 'arrow-right';
icon.size = 24;
icon.color = 'color-feedback-success-text';
document.body.appendChild(icon);

// Dynamic icon switching
const statusIcon = document.querySelector('icon-base');
statusIcon.name = 'check';    // Change icon
statusIcon.size = 32;          // Change size
statusIcon.color = '#4ECDC4';  // Change color

// Wait for component registration
await customElements.whenDefined('icon-base');
console.log('icon-base is ready');
