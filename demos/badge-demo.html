Read-only, non-interactive visual indicators for status, categorization, and notification counts

1. Badge Family Overview

The Badge family consists of three components for different use cases:

  • Badge-Label-Base = Text labels for categorization, status, metadata
  • Badge-Count-Base = Numeric counts with circular/pill shape
  • Badge-Count-Notification = Notification counts with live region announcements

Notification Alerts
Key Characteristic: All badges are read-only and non-interactive. They don't respond to clicks and are not focusable.

11. Token Verification

Check browser DevTools to verify these CSS custom properties are loaded:

Typography Tokens

  • --typography-label-xs - Small badge text
  • --typography-label-sm - Medium badge text
  • --typography-label-md - Large badge text

Spacing Tokens

  • --space-inset-050 - Minimal padding
  • --space-inset-100 - Standard padding
  • --space-inset-150 - Large padding
  • --space-grouped-minimal - Icon gap (sm)
  • --space-grouped-tight - Icon gap (md, lg)

Color Tokens

  • --color-badge-notification-background - pink400 (#CC2257)
  • --color-badge-notification-text - white100 (#FFFFFF)
  • --color-surface - Default badge background
  • --color-text-default - Default badge text

Radius Tokens

  • --radius-subtle - Label badge corners (2px)
  • --radius-full - Count badge circular/pill (9999px)

12. Usage Examples

HTML

<!-- Basic label badge -->
<badge-label-base label="Draft"></badge-label-base>

<!-- Label badge with icon -->
<badge-label-base label="Approved" icon="check" size="sm"></badge-label-base>

<!-- Label badge with truncation -->
<badge-label-base label="Very long text" truncate="true"></badge-label-base>

<!-- Count badge -->
<badge-count-base count="5"></badge-count-base>

<!-- Count badge with custom max -->
<badge-count-base count="150" max="50"></badge-count-base>

<!-- Notification badge -->
<badge-count-notification count="12"></badge-count-notification>

<!-- Notification badge without announcements -->
<badge-count-notification count="5" announce-changes="false"></badge-count-notification>

JavaScript

// Programmatic usage - Label badge
const labelBadge = document.createElement('badge-label-base');
labelBadge.label = 'Status';
labelBadge.size = 'sm';
labelBadge.icon = 'check';
document.body.appendChild(labelBadge);

// Programmatic usage - Count badge
const countBadge = document.createElement('badge-count-base');
countBadge.count = 42;
countBadge.max = 99;
document.body.appendChild(countBadge);

// Programmatic usage - Notification badge
const notificationBadge = document.createElement('badge-count-notification');
notificationBadge.count = 5;
notificationBadge.announceChanges = true;
document.body.appendChild(notificationBadge);
