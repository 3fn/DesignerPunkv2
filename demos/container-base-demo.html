1. Overview

Container-Base is the foundational primitive for all container components. It wraps child content with configurable padding, background color, border, shadow, and border-radius ‚Äî all mapped to design system tokens.

3. Directional Padding

Container-Base supports a three-level padding override hierarchy using CSS logical properties. Individual edges override axis props, which override uniform padding.

Axis Props (paddingVertical / paddingHorizontal)

Tight vertical (8px), wide horizontal (32px)

 padding-vertical="100" padding-horizontal="400"

Wide vertical (32px), tight horizontal (8px)

 padding-vertical="400" padding-horizontal="100"

Individual Edge Props

Large top (32px), small bottom (4px), medium sides (16px)

 Individual edge control

Uniform 100 (8px) with block-start override to 400 (32px)

 Override hierarchy demo
Override hierarchy: Individual edges (highest) ‚Üí Axis props ‚Üí Uniform padding (lowest). All use CSS logical properties for RTL support.

11. Token Verification

Check browser DevTools to verify these CSS custom properties are loaded:

Spacing Tokens (Padding)

  • --space-inset-050 ‚Äî padding 050 (4px)
  • --space-inset-100 ‚Äî padding 100 (8px)
  • --space-inset-150 ‚Äî padding 150 (12px)
  • --space-inset-200 ‚Äî padding 200 (16px)
  • --space-inset-300 ‚Äî padding 300 (24px)
  • --space-inset-400 ‚Äî padding 400 (32px)

Color Tokens

  • --color-surface ‚Äî default surface background
  • --color-canvas ‚Äî canvas background
  • --color-background ‚Äî page background
  • --color-border-default ‚Äî default border color
  • --color-action-primary ‚Äî primary accent border

Shadow Tokens

  • --shadow-container ‚Äî container elevation shadow
  • --shadow-navigation ‚Äî navigation elevation shadow
  • --shadow-dropdown ‚Äî dropdown elevation shadow
  • --shadow-modal ‚Äî modal elevation shadow

Border Tokens

  • --border-default ‚Äî default border width (1px)
  • --border-emphasis ‚Äî emphasis border width (2px)
  • --border-heavy ‚Äî heavy border width (4px)

Radius Tokens

  • --radius-050 ‚Äî tight radius (4px)
  • --radius-100 ‚Äî normal radius (8px)
  • --radius-200 ‚Äî loose radius (16px)

Motion Tokens

  • --motion-focus-transition-duration ‚Äî hover transition duration
  • --motion-focus-transition-easing ‚Äî hover transition easing

12. Usage Examples

HTML

<!-- Basic container -->
<container-base padding="200" background="color-surface" border-radius="normal">
  <p>Content goes here</p>
</container-base>

<!-- With shadow and border -->
<container-base
  padding="300"
  background="color-surface"
  shadow="shadow-container"
  border="default"
  border-radius="loose"
>
  <p>Elevated content</p>
</container-base>

<!-- Custom border color -->
<container-base
  padding="200"
  background="color-surface"
  border="emphasis"
  border-color="color-action-primary"
  border-radius="normal"
>
  <p>Accented container</p>
</container-base>

<!-- Directional padding -->
<container-base
  padding-vertical="100"
  padding-horizontal="300"
  background="color-surface"
  border-radius="normal"
>
  <p>Asymmetric padding</p>
</container-base>

<!-- Semantic HTML with accessibility -->
<container-base
  padding="200"
  semantic="article"
  accessibility-label="Blog post"
  background="color-surface"
  border-radius="normal"
>
  <h2>Article Title</h2>
  <p>Article content</p>
</container-base>

<!-- Hoverable container -->
<container-base
  padding="200"
  background="color-surface"
  border-radius="normal"
  hoverable="true"
>
  <p>Hover me!</p>
</container-base>

JavaScript

// Programmatic creation
const container = document.createElement('container-base');
container.setAttribute('padding', '200');
container.setAttribute('background', 'color-surface');
container.setAttribute('border-radius', 'normal');
container.setAttribute('shadow', 'shadow-container');
container.innerHTML = '<p>Dynamic content</p>';
document.body.appendChild(container);

// Update attributes dynamically
container.setAttribute('border', 'emphasis');
container.setAttribute('border-color', 'color-action-primary');

// Semantic container
const article = document.createElement('container-base');
article.setAttribute('semantic', 'article');
article.setAttribute('accessibility-label', 'Blog post');
article.setAttribute('padding', '300');
article.setAttribute('background', 'color-surface');
article.innerHTML = '<h2>Title</h2><p>Content</p>';
document.body.appendChild(article);
