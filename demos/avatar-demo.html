Visual representation component for users (Human) and AI agents (Agent) with shape-based differentiation

1. Entity Type Differentiation

Avatars use shape to instantly distinguish between human users and AI agents:

  • Human = Circle (organic, natural)
  • Agent = Hexagon (synthetic, constructed)
Human Circle Shape
Agent Hexagon Shape
Accessibility: Shape differentiation doesn't rely on color alone, improving accessibility for colorblind users.

7. Accessibility Features

Decorative Mode

When decorative="true", avatar is hidden from screen readers (aria-hidden="true").

decorative="true" aria-hidden
decorative="false" announced

Test ID Support

Use testid attribute for automated testing (renders as data-testid).

testid="user-profile-avatar"

Alt Text for Images

When src is provided, alt is required for accessibility.

alt="John Smith's profile picture"

8. Default Values

When props are omitted, these defaults apply:

No props type="human", size="md"
Only type size="md" (default)
Only size type="human" (default)
  • type ‚Üí defaults to "human"
  • size ‚Üí defaults to "md"
  • interactive ‚Üí defaults to false
  • decorative ‚Üí defaults to false

9. Token Verification

Check browser DevTools to verify these CSS custom properties are loaded:

  • --avatar-size-xs = 24 (unitless)
  • --avatar-size-sm = 32 (unitless)
  • --avatar-size-md = 40 (unitless)
  • --avatar-size-lg = 48 (unitless)
  • --avatar-size-xl = 80 (unitless)
  • --avatar-size-xxl = 128 (unitless)
  • --avatar-icon-size-xs = 12 (unitless)
  • --avatar-icon-size-xxl = 64 (unitless)
  • --color-avatar-human = orange300 reference
  • --color-avatar-agent = teal300 reference
  • --color-avatar-border = gray100 reference
  • --motion-focus-transition-duration = 150ms

10. Usage Examples

HTML

<!-- Basic human avatar -->
<avatar-base type="human" size="md"></avatar-base>

<!-- Agent avatar with interactive hover -->
<avatar-base type="agent" size="lg" interactive="true"></avatar-base>

<!-- Human avatar with image -->
<avatar-base
  type="human"
  size="xl"
  src="/profile.jpg"
  alt="User profile">
</avatar-base>

<!-- Decorative avatar (hidden from screen readers) -->
<avatar-base type="human" size="sm" decorative="true"></avatar-base>

JavaScript

// Programmatic usage
const avatar = document.createElement('avatar-base');
avatar.type = 'agent';
avatar.size = 'lg';
avatar.interactive = true;
document.body.appendChild(avatar);
