TextInputField - Actual Component Preview

This preview uses the actual <text-input-field> web component implementation.

Default State

Empty input, not focused. Label appears inside the input field.

Focused State

Click the input to see the label animate to floated position.

Filled State

Input with content. Label remains in floated position.

Error State

Input with validation error. Shows error icon and error message.

Success State

Input with successful validation. Shows success icon (checkmark).

Helper Text

Input with persistent helper text below the field.

Helper Text + Error State

Both helper text and error message visible simultaneously.

Required Field

Input marked as required.

Read-Only State

Input in read-only mode (alternative to disabled).

Info Icon

Input with informational icon.
