Simplified Primitive Token Structure

One value per quality level for clarity

Token Name Value Usage
Shadow Offset Primitives
shadowOffsetX100 4px Small horizontal offset
shadowOffsetX150 6px Medium horizontal offset (strategic flexibility)
shadowOffsetX200 8px Standard horizontal offset
shadowOffsetX300 12px Large horizontal offset
shadowOffsetY100 4px Small vertical offset (elevation 1)
shadowOffsetY200 8px Medium vertical offset (elevation 2)
shadowOffsetY300 12px Large vertical offset
shadowOffsetY400 16px Extra large vertical offset (elevation 3)
Shadow Blur Primitives (Quality Levels)
shadowBlurHard 4px Hard shadow - sharp, defined edges
shadowBlurModerate 12px Moderate shadow - balanced definition
shadowBlurSoft 20px Soft shadow - diffuse, gentle edges
Shadow Opacity Primitives (Quality Levels)
shadowOpacityHard 0.4 Hard shadow - darker, more defined
shadowOpacityModerate 0.3 Moderate shadow - balanced visibility
shadowOpacitySoft 0.2 Soft shadow - lighter, more subtle
Depth-Based Adjustments
shadowBlurDepth200 16px Moderate blur for depth 200 (2× scale)
shadowBlurDepth300 24px Soft blur for depth 300 (3× scale)
shadowOpacityDepth200 0.35 Slightly darker for depth 200
shadowOpacityDepth300 0.4 Darker for depth 300
